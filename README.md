# Smart Exam Planner, Revision Tracker & AI Study Assistant

## Overview
A desktop-based application built using Python and Tkinter to help students:
- Plan study topics
- Track revision progress
- Identify weak areas
- Analyze performance
- Generate AI-based study notes

## Features
- Add, update, delete study topics
- Status tracking (Pending / Revising / Done)
- Priority management
- Weak area detection (based on deadlines & priority)
- Analytics dashboard (completion %, overdue topics)
- Visual charts (pie charts, bar charts)
- AI-based study assistant (requires API key)

## Tech Stack
- Python
- Tkinter
- Pandas
- Matplotlib
- Excel (backend storage)

## How to Run
```bash
pip install pandas matplotlib openai
python main.py
